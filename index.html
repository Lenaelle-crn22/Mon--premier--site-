C'est quoi une page web statique?

Une page web statique est une page web dont le contenu ne varie pas en fonction des caracteristiques de la demande. C‘est a dire qu‘a un moment donnée tout les internautes qui demandent la page recoivent le meme contenu

Quels language permettent de rendre une page web statistique interactive?

Le langage qui permet de rendre une page Web statique interactif, et JavaScript

C'est quoi une page web dynamique?

Une page Web dynamique est une page Web générée à la demande,par opposition à une page Web stative. Le contenu d'une page Web dynamique peut varier en fonction d'information(heure,nom de l'utilisateur) qui sont connues qu'au moment de sa consultation .A l'inverse ,le contenu d'une page web statique est opignion identique. a chaque consultation.
